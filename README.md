#  CodeGuard AI - Context-Aware Security Auditor

**CodeGuard AI** is a professional, multi-agent security tool designed to analyze source code for vulnerabilities. Powered by **Llama 3.3 (70B)** via Groq, it doesn't just find bugs—it understands the **context** of your code to provide meaningful, prioritized security insights.

##  Key Features

* **Context-Aware Auditing:** Smart analysis that distinguishes between a simple "Calculator" script and a complex "Database Management" system.
* **Executive Dashboard:** Real-time visual summaries with interactive charts (Safe vs. Vulnerable).
* **Multi-File Support:** Upload and scan multiple files (C++, Python, JS, etc.) simultaneously.
* **Cyber-Security UI:** A sleek, dark-themed interface built for security professionals.
* **Secure API Handling:** Personal API keys are handled via Session State and never stored insecurely.

---

##  Tech Stack

* **Frontend:** [Streamlit](https://streamlit.io/)
* **AI Engine:** [Groq Cloud](https://groq.com/) (Llama-3.3-70b-versatile)
* **Visuals:** Plotly & Streamlit Lottie
* **Logic:** Multi-agent prompt engineering for security assessment.
